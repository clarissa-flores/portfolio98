## Scripts

- `npm run dev` – start Astro dev server
- `npm run build` – production build
- `npm run preview` – preview the built site
- `npm run check` – Astro type/syntax checks

## Assets & Licenses

- `public/solitaire/` contains the bundled js-solitaire assets; see `licenses/LICENSE-solitaire.txt`.
- `public/oneko/` contains the oneko assets; see `licenses/LICENSE-oneko.txt`.
- `licenses/WINBOX-LICENSE.txt` documents WinBox usage.

Keep third-party assets in `public/<asset>/` and add their licenses in `licenses/` with a matching name.
